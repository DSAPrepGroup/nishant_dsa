Explanation

  1. The function is_valid_palindrome takes a string as input.
  2. It first cleans the string by converting it to lowercase and removing all non-alphanumeric characters.
  3. Then it compares the cleaned string with its reverse using string slicing ([::-1]).
  4. If they are equal, it's a valid palindrome, and the function returns True. Otherwise, it returns False.
